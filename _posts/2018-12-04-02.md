<br>
```swift
import Foundation

@objc class Printer: NSObject {

    func doPrint() {
        print("PRINT!")
    }

}
```
"@objc" and "NSObject" is the key point.<br>
<br>
```swift
#import "YourProjectName-Swift.h"
```
Add import to Objective-c file, then you can use Wwift in Objective-c.
